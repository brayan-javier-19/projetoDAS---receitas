# Página Colaborativa de Receitas

Este projeto foi desenvolvido no âmbito da unidade curricular de **Desenvolvimento Ágil de Software**.

O objetivo é criar uma página web colaborativa onde os elementos do grupo podem adicionar e partilhar receitas de bolos, utilizando boas práticas de desenvolvimento e controlo de versões através do Git e GitHub.

## Objetivo do Trabalho

Este projeto permite aplicar conceitos importantes de desenvolvimento colaborativo, tais como:

- Utilização do Git para controlo de versões;
- Utilização do GitHub para alojamento e colaboração no projeto;
- Criação e gestão de branches;
- Realização de commits;
- Utilização de Pull Requests para integração de alterações.

Cada elemento do grupo contribui para a evolução da página através da criação e adição de novas receitas.

## Execução do Projeto

Para executar o projeto basta abrir o ficheiro `index.html` num navegador web.

A página apresenta várias receitas organizadas por cartões, contendo:

- Nome da receita;
- Lista de ingredientes;
- Modo de preparação;
- Informações adicionais quando necessário.

A navegação entre as receitas é realizada através do menu existente no topo da página, permitindo aceder rapidamente a cada receita disponível.
